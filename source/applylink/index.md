## 申请要求

{% note default flat %}
1. **内容健康**：网站内容必须健康向上，不包含任何违法、违规、低俗或侵权内容。
2. **原创内容**：优先考虑原创内容丰富的网站，转载内容需注明出处。
3. **定期更新**：网站需保持定期更新，长期不更新的网站将不予通过。
4. **互相链接**：请先在您的网站上添加本站的友链信息，然后提交申请。
{% endnote %}
